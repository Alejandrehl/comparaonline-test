# ComparaOnline Test

[Test Requirements](https://github.com/comparaonline/interview-coding-test)

## Getting Started

1. Clone repo:
   - git clone git@github.com:Alejandrehl/comparaonline-test.git
2. Go to project folder:
   - cd comparaonline-test
3. Install packages;
   - npm install
4. Exec script:
   - npm run after-30-days

## Running the tests

    - npm run test
