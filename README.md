# ComparaOnline Test

## Getting Started

    - Clone repo: git clone git@github.com:Alejandrehl/comparaonline-test.git
    - Go to folder: cd comparaonline-test
    - Install packages: npm install
    - Exec script: npm run after-30-days

## Running the tests

    - Exec test: npm run test
